## Goal

`.design/tasks/` の `owner: codex` タスクを、完了条件に直接必要な範囲だけ実装する。

## Process

1. 対象タスクファイルを読み、`## 概要` と `## 完了条件` だけを先に確定する。
2. 変更対象ファイルを最小化する（完了条件達成に不要な変更はしない）。
3. 実装する。
4. タスク記載の検証コマンドがあれば実行し、結果を報告する。
5. `/close` へ引き継ぐため、変更ファイルと完了条件の達成状況を整理して報告する。

## Guardrails

- スコープを拡張しない。
- リファクタリングや最適化は、タスクに明記されている場合のみ行う。
- 疑義がある要件は推測せずユーザー確認する。
- タスクに書かれていない仕様追加をしない。
